# MapBiomas Venezuela - Coverage

Scripts for classifying land use and land cover across Venezuela. Uses Google Earth Engine to apply the official MapBiomas mapping methodology across all biomes and regions.

## Getting Started

To run the scripts you will need:

- A [Google Earth Engine](https://earthengine.google.com/) account.
- Access to the required assets (some are public, others must be copied to your account).

Follow the instructions in this repository's README.md to set output paths and required modules.
